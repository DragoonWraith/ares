Ares Editor

Ares Editor is an open-source 3D game creation kit for Crystal Space and Crystal Entity Layer.

This document includes usage instructions, conceptual material, and detailed technical information.

This manual, edition 0.0.1, is for AresEd, version 0.0.1, and was last updated on 31 October 2012 .
