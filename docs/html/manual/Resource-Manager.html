2.3 Resource Manager

With the resource manager you can manage the individual resources (like dynamic factories, templates, light factories and quests). You can move resources between assets, delete resources, purge unused resources and so on.

The resource manager also attempts to show how many times each resource has been used. While this works in most situations it is possible that this is not 100% accurate. The resource usage counter doesn't take variables and parameters into account and so it is possible that (for example) to have a template that apparently isn't used but it is still needed because the template name itself is given as a parameter through some other mechanism. If you want to keep using the option to remove unused resources then one way to solve this problem is to LOCK the resource. Locked resources will never be removed accidently.
