4.1 Moving the Camera

There are many ways to move the camera in the 3D view.

Using the middle mouse button you can select a point in the world and then pan the camera in 3D around that point. If you do this in combination with the ‘shift’ key you will slide the world.

Using the mouse wheel you can zoom in towards the point where the mouse points or away from that point.

It is also possible to move the camera using the classical ‘WASD’ keys. Use this in combination with ‘Ctrl’ to slow down movement. Use this in combination with ‘Shift’ to pan the view. ‘PgDn’ and ‘PgUp’ are also usable to look down/up. If ‘Pan’ is enabled in the camera panel below the object list then these keys act differently if an object is selected. In that case the camera will always look at the selected object(s) and the ‘WASD’ keys will simply pan around that object.

If you have an object selected you get various tools to reposition the camera based on that selection:

  • ‘Look at selection’: Look at the selected object(s) without moving the camera. This functionality is also present as a button (‘Look’) on the left side below the object list.
  • ‘Move to selection’: With this option you can move the camera near the selected object(s) and reposition the camera so that it faces those objects. This functionality is also present as a button (‘Move’) on the left side below the object list.
  • ‘Top of selection’: Move the camera above the selection, looking down.

In addition you can also select ‘Top of the world’ which will move the camera very high above the world.

Using the camera panel below the object list there are also some other things you can do. Using the ‘N’, ‘S’, ‘E’, and ‘W’ buttons you can orient the camera directly towards a specific direction without changing position.

Using the three ‘S’ and ‘R’ keys you can ‘Store’ and ‘Remember’ a camera location.
