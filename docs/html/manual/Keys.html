6. Keys

Here is a list of all keys supported by AresEd:

  • ‘1’: Toggle debug mode.
  • ‘2’: Toggle static/dynamic state of current selected object(s).
  • ‘F2’: Increase the current time for the nature system.
  • ‘F3’: Toggle automatic time progression for the nature system.
  • ‘F4’: Decrease foliage density factory.
  • ‘F5’: Increase foliage density factory.
  • ‘p’: Toggle panning around selected object(s) (equivalent to the ‘pan’ checkbox that you can find on the camera panel on the left side).
  • ‘l’: Toggle display of labels in 3D view.
  • ‘e’: Spawn a new item from the currently selected factory in the factory tree.
  • ‘g’: Start grabbing the current selected object(s) (movement).
  • ‘q’: While grabbing, pasting, or spawning a new item this toggles grid mode.
  • ‘x’: While grabbing, pasting, or spawning a new item this toggles X-axis constraint.
  • ‘y’: While grabbing, pasting, or spawning a new item this toggles Y-axis constraint.
  • ‘z’: While grabbing, pasting, or spawning a new item this toggles Z-axis constraint.
  • ‘Up’: Move the selected object(s) along the world Z-axis (‘Ctrl’ to slow down, ‘Shift’ to speed up).
  • ‘Down’: Move the selected object(s) along the world Z-axis (‘Ctrl’ to slow down, ‘Shift’ to speed up).
  • ‘Left’: Move the selected object(s) along the world X-axis (‘Ctrl’ to slow down, ‘Shift’ to speed up).
  • ‘Right’: Move the selected object(s) along the world X-axis (‘Ctrl’ to slow down, ‘Shift’ to speed up).
  • ‘<’ or ‘,’: Move the selected object(s) along the world Y-axis (‘Ctrl’ to slow down, ‘Shift’ to speed up).
  • ‘>’ or ‘.’: Move the selected object(s) along the world Y-axis (‘Ctrl’ to slow down, ‘Shift’ to speed up).
  • ‘ESC’: Use this key to exit play mode.
  • ‘w’, ‘a’, ‘s’, ‘d’: If panning is disabled these keys control WASD movement for the camera. With panning enabled these keys move around the selected object(s). (‘Ctrl’ to slow down, ‘Shift’ to speed up).
  • ‘PgDn’, ‘PgUp’: If panning is disabled these keys look up/down with the camera. With panning enabled these keys move around the selected object(s). (‘Ctrl’ to slow down, ‘Shift’ to speed up).
